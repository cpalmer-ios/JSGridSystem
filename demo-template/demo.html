JSGrid System

Create Grid

add your basic HTML setup:

            //index.html
            <div id="grid" class="jsgrid_grid grid" > 
              <div class="jsgrid_item"></div>
              <div class="jsgrid_item"></div>
              <div class="jsgrid_item"></div> 
          </div>
          (jsgrid_items must be equal to the number of columns defined.)

Import your grid preferences from JSON

            //main.js

            ...

            const JSON =  {
               "id": "#grid",               - ID of the grid element
               "dimensions": {    
                 "height": "200",           - height in pixels
                 "margin": "0",             - margin
                 "minHeight": "100",        - minHeight per column
                 "maxHeight": "300"         - maxHeight per column
               },
               "animations": {              - animation attributes
               "fadeInSpeed": "500"         - fade in speed
               },
               "style": {
                 "background": "random",    - creates random background colours
                 "borderRadius": "5"        - border styling
               },
               "responsive": [
                 {
                   "breakpoint": 1440,      - set screen breakpoints
                   "columns": 12            - set max columns here
                 }
               ], 
               "blocks": [8, 2, 2]          - define block configuration here ( must equal 12)
           }

            ...

Or just use HTML / CSS:

        //index.html
        <div class="jsgrid_grid grid" > 
          <div class="jsgrid_grid_row" > 
            <div class="jsgrid_block w-8"></div>
            <div class="jsgrid_block w-4"></div>
        </div>
      </div>
      width is set with the w-{number} class. ie. w-1 is 1 of 12
